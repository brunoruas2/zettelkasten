# Unique Identifier
202510290617

# Tag
#DesignPattern 

# Body
Criar uma interface para criação de objetos mas deixar que as subclasses decidam qual objeto criar. Assim podemos garantir **como** os objetos serão criados sem nos preocuparmos com **quando**.
